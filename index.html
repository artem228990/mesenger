Добро пожаловать в SwapLines — инновационную криптовалютную биржу с передовыми инструментами для трейдинга и анализа рынка. Наша платформа обеспечивает безопасность, скорость и удобство работы с цифровыми активами.
Регистрация и Верификация
Для начала работы зарегистрируйтесь, подтвердите e-mail и пройдите процедуру KYC для повышения лимитов и доступа ко всем функциям.
API Доступ
Наш REST API позволяет:
  • Получать котировки и исторические данные
  • Создавать и отменять ордера
  • Управлять балансом и выводить средства
  • Отслеживать статусы сделок и уведомления
Для получения API ключа перейдите в личный кабинет в раздел "API".
